Layouts Fluídos/2

Fitros e manipulação com CSS

Os filtros são usados para colocar efeitos em imagens e edita-las sem precisar usar editor de imagem


Alguns exemplos de filtros:


  • Greyscale : Coloca filtro de escala cinza na imagem;
  • Blur : Adiciona um efeito desfocado;
  • Brightness : Adiciona brilho a imagem;
  • Drop-shadow : Adiciona sombreamento na imagem;

Para o layout se tornar fluído utiliza-se as propriedades width e height nas seguintes medidas relativas:

  • porcentagem (%)
  • viewport width (vw)
  • viewport height (vh)

Essas unidades permitem que os elementos se adaptem ao tamanho da tela do usuário.

Para configurar uma largura fixa, pode criar uma .class e definir o width utilizando unidades absolutas (px). Por exemplo:
