<div>

  <p>Para configurar uma <u>largura fixa</u>, pode criar uma <i>.class</i> e definir o <i>width</i> utilizando <u>unidades absolutas (px)</u>. Por exemplo:</p>

  <p class="largurafixa">Largura Fixa</p>

 </div>


.largurafixa {

    width: 200px;

    margin: 40px;

    text-align: center;

    padding: 20px;

    background-color: black;

    color: white;

}
