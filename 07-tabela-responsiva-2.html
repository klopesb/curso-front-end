Título da tabela ( thead )

Para colocar o titulo da tabela no cabeçalho, foi colocado em uma div as tags de cabeçalho table row e table head com uma .class identificando o titulo da tabela.

Também no cabeçalho da tabela, para separar os dados dos gastos de acordo com os meses do ano, foi usada as mesmas tags (tr e th), mas sem o .class. ara diferenciar o cabeçalho das outras células em CSS, foi utilizado os seletores table + th.

Corpo da tabela ( tbody )

Rodapé da tabela ( tfoot )

O rodapé da tabela é onde vai estar os valores totais da soma dos gastos de todas as contas por mês, portanto ela estará destacada de forma diferente do corpo da tabela.
