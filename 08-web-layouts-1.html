Layouts Fluídos

Dimensões CSS

Os layouts fluídos são aqueles que acompanham o tamanho da tela, apenas aumentando e diminuindo usando porcentagens para definir a area. Fonte: TurboSite.


As propriedades de dimensões do CSS são:


  • width : Configura a largura do elemento;
  • max-width : Configura a largura máxima do elemento;
  • min-width : Configura a largura mínima do elemento;
  • height : Configura a altura do elemento;
  • max-height : Configura a altura máxima do elemento;
  • min-width : Configura a largura mínima do elemento.

Para o layout se tornar fluído utiliza-se as propriedades width e height nas seguintes medidas relativas:

  • porcentagem (%)
  • viewport width (vw)
  • viewport height (vh)

Essas unidades permitem que os elementos se adaptem ao tamanho da tela do usuário.
